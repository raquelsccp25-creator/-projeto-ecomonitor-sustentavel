# EcoMonitor 4.0 🌿

Projeto desenvolvido para o curso de TI com foco em **Indústria Sustentável** e **Manutenção Preditiva**.

## 🚀 Objetivo
O objetivo deste projeto é demonstrar como o monitoramento em tempo real (temperatura, tensão e precisão) pode evitar o desperdício de matéria-prima e reduzir o consumo excessivo de energia na indústria.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Estrutura Semântica)
* CSS3 (Layout Flexbox e Design Responsivo)
* GitHub (Controle de versão e colaboração)

## 📋 Funcionalidades
- Monitoramento de temperatura com alertas de segurança.
- Status de eficiência das máquinas (Bom, Alerta, Crítico).
- Interface visual intuitiva para operadores de fábrica.
